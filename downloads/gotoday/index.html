MVP Scope

첫 출시는 추천 5개를 정확히 만드는 데 집중

MVP에 포함

  • 현재 위치 가져오기
  • 동행, 시간, 예산, 실내/실외 조건 선택
  • 날씨와 미세먼지 조회
  • 관광/장소 후보 수집
  • 거리 계산과 점수 계산
  • 추천 이유가 포함된 결과 카드
  • 외부 지도 앱 연결

MVP에서 제외

  • 로그인과 소셜 계정 연동
  • AI 추천과 AI 요약
  • 실시간 혼잡도
  • 복잡한 코스 자동 생성
  • 사용자 리뷰/커뮤니티
  • 결제와 광고
  • 고도화된 관리자 페이지

User Flow

사용자는 검색하지 않고 조건만 고른다

  1. 위치 확인 현재 위치 사용 권한을 받고 주변 후보를 찾는다.
  2. 상황 선택 혼자, 연인, 친구, 아이와, 부모님 중 동행 유형을 고른다.
  3. 제약 선택 이동 가능 시간, 예산, 실내/실외, 반려동물, 유모차 조건을 고른다.
  4. 추천 계산 거리, 날씨, 미세먼지, 예산, 동행 적합도를 점수화한다.
  5. 결과 확인 추천 5개와 이유를 보고 지도 앱으로 이동한다.

Architecture

혼자 운영 가능한 작은 서버 구조

Flutter

화면, 위치 권한, 조건 입력, 결과 표시, 지도 연결만 담당한다.

features/recommendation

FastAPI

외부 API 키 보호, 후보 수집, 점수 계산, 캐시 저장을 담당한다.

services/scoring_service.py

PostgreSQL

장소 보정 데이터, 추천 로그, 즐겨찾기, 피드백을 저장한다.

places, recommendation_logs

Mini PC + WSL

초기 서버로 사용하되, 외부 공개와 백업 정책은 별도로 관리한다.

Docker Compose

초기 API

  • POST /recommendations 조건 기반 추천 요청
  • GET /places/{place_id} 장소 상세 조회
  • POST /favorites 즐겨찾기 저장
  • POST /feedback 추천 품질 피드백 저장
  • GET /health 서버 상태 확인

External APIs

초기에는 API 수를 줄이고 비용을 통제

용도 1순위 비고
장소 후보 한국관광공사 TourAPI 관광지, 행사, 문화시설 후보 확보
지도/장소 검색 카카오 로컬 API 좌표, 키워드, 카테고리 검색에 사용
